Una tienda online a medida para Fahrzeugtechnik Rester basada en el CMS Redaxo. Por desgracia, es el único proyecto que nunca se llegó a publicar.

La tienda no se pudó lanzar con un sistema como Shopify por la complejidad de las bases de datos. Cada proveedor de productos tenía otro sistema de bases de datos lo que causó que la página era demasiado lenta.
Aunque hemos indicado al cliente que la rapidéz de la página web no sería adecuada y las bases de datos necesitaban ser optimizadas, el cliente decidió continuar sin hacer estas mejoras.
Esto resultó en que el sitio web realmente no era usable y la tienda no se desplegó hasta que los cambios necesarios eran realizados. La versión actual fue hecha por otra agencia.
