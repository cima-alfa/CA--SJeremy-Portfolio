Para Waffenversand Klausing hicimos un sitio web multilingual a medida con un sistema de pedidos de transporte de armas legal - nacional e internacional.

Integramos el XML API de DHL para crear etiquetas y pedir fechas de recogida para los clientes. Al principio el sistema de pago era simplemente generar una factura y hacer transferencia bancaria.
Con tiempo añadimos métodos de pago más avanzados como tarjetas de crédito, PayPal Plus, Klarna y PayMorrow junto con una calculadora de precios.

Los clientes también tenían la opción de registrarse para tener sus datos guardados y poder ver el status de sus pedidos.
En la administración, era posible ver, editar y exportar los pedidos en formato CSV para poder subirlos al sistema de DHL.
