La página de Autobasar Amir necesitó un sistema a medida para ofrecer un servicio de evaluar el precio de un coche de segunda mano.

Para cargar toda la información de los coches que existen usamos el XML API de mobile.de. El cliente selecciona su coche, añade información, fotos y una vez enviado, recibirá la oferta por parte de Autobasar Amir.
