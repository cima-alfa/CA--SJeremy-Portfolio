La versión AMP (Accelerated Mobile Pages) de Waffenversand Klausing sirvió como una landing page con enlaces a la versión normal de la página.

AMP es una herramienta para hacer un sitio web más rápido. Se necesitan menos recursos para cargar la página web - esto permite lograr una mejor posición en los motores de búsqueda, aunque hoy en día ya no se suele usar por los avances de las tecnologías y es posible que dentro de poco AMP se va a quedar obsoleto.
