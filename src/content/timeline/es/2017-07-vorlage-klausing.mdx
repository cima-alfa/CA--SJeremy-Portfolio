Waffenversand Klausing: Vorlage era un sistema a medida para generar HTML para la página de subastas de armas egun.de.

El usuario añade un arma al sistema junto con la información y fotos. El sistema genera HTML que se puede copiar y pegar en la página egun.de.
Además el contenido generado se puede ver y editar. El HTML tiene que seguir ciertas reglas de egun.de, no es posible usar técnicas modernas ni responsivas.
